# The Thinking Athlete — Chess Content Pipeline 🏃♟️

Automated pipeline: chess.com wins → animated video → YouTube → Instagram

---

## What This Does

1. **Step 1** — Fetches your last 3 months of games from chess.com, finds your wins, picks the most interesting ones (by rating difference)
2. **Step 2** — Generates an animated MP4 video of the game with your branding overlay
3. **Step 3** — Uploads the video to YouTube with auto-generated funny title + description
4. **Step 4** — Posts to Instagram as a Reel (optional, needs Facebook Business setup)

---

## Setup

### Install dependencies
```bash
pip install chess cairosvg imageio imageio-ffmpeg pillow requests google-auth google-auth-oauthlib google-api-python-client
```

### For YouTube upload (Step 3):
1. Go to https://console.cloud.google.com
2. Create project → Enable **YouTube Data API v3**
3. Create **OAuth 2.0 credentials** → Desktop App
4. Download as `client_secrets.json` → place in this folder
5. First run opens browser to authenticate — automatic after that

### For Instagram (Step 4):
1. You need a **Facebook Business account** + **Instagram Professional account**
2. Go to https://developers.facebook.com → Create App → Instagram Graph API
3. Get a long-lived access token
4. Fill in `ACCESS_TOKEN` and `INSTAGRAM_ACCOUNT_ID` in step4_post_instagram.py

---

## Run the Pipeline

```bash
# Step 1 — Fetch your won games from chess.com
python step1_fetch_games.py

# Step 2 — Generate video (edit GAME_INDEX to pick which game)
python step2_generate_video.py

# Step 3 — Upload to YouTube (set to private first, review, then make public)
python step3_upload_youtube.py

# Step 4 — Post to Instagram (optional)
python step4_post_instagram.py
```

---

## Tips

- **Review before publishing** — Step 3 uploads as `private` by default. Watch it, add your voiceover in any editor (CapCut, DaVinci Resolve), then make it public.
- **Change GAME_INDEX** in step2 to generate videos for different games (0 = most interesting, 1 = second most, etc.)
- **Run weekly** — every Sunday, run step1 + step2 to generate content for the week
- **Add voiceover** — the video is silent by default. Record a 60-second commentary in CapCut and layer it over the video before uploading.
